# Retail Sales Data Analyzer

## Project Overview

The **Retail Sales Data Analyzer** is a Python-based application designed to process, analyze, and visualize retail sales data. This project demonstrates the use of control structures, object-oriented programming (OOP), NumPy, Pandas, and data visualization libraries like Matplotlib and Seaborn.

## Objectives

* Load and validate retail sales data from a CSV file
* Perform data cleaning and analysis
* Calculate key sales metrics
* Filter data based on user input
* Visualize trends using graphs and charts


## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn

##Project Structure

##  Dataset Details

The dataset (`practicalexam.csv`) contains the following columns:

* `Date` (YYYY-MM-DD)(in this structure date is mentioned)
* `Product`
* `Category`
* `Price`
* `Quantity Sold`
* `Total Sales`


## Features

* Data loading and validation
* Handling missing values
* Sales metrics calculation (total, average, popular product)
* Data filtering (by category and date range)
* Data visualization:

* Bar Chart (Sales by Category)
* Line Graph (Sales Trend)
* Heatmap (Correlation)
